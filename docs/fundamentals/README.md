# Fundamentals Notes Index

This directory contains milestone theory notes written for learners and new contributors.

Use these notes to answer two practical questions quickly:

1. What is happening in this milestone at runtime?
2. Why is this scope chosen now, and what is intentionally deferred?

## Current Notes

- `docs/fundamentals/M1-boot-to-rust.md`: Step-by-step explanation of the M1 UEFI to Rust execution path in QEMU/OVMF.
- `docs/fundamentals/milestone-theory-template.md`: Reusable template for future milestone theory notes.

## Authoring Expectations

- Keep language friendly and concrete.
- Prefer execution-path explanations over abstract architecture talk.
- Link every major concept to a real file, script, or command in this repository.
- Keep each milestone theory note aligned with:
  - `docs/specs/<milestone>/prd.md`
  - `docs/specs/<milestone>/design.md`
  - `docs/reviews/<milestone>/review.md` (when available)
